隱私權政策

最後更新：2026-02-22

關於本站

本站（marsantony.github.io）及其子專案由 Mars Liu 個人經營維護。 本站並非營利網站，所蒐集之資料僅供網站管理者個人使用。

資料蒐集

本站使用自建的輕量分析系統來了解網站流量與改善內容。此系統：

  • 不使用 Cookie
  • 不蒐集姓名、Email 或任何可直接識別個人的資訊
  • 不追蹤跨站瀏覽行為

蒐集的匿名資料

  • 瀏覽的頁面路徑
  • 來源頁面（referrer）
  • 瀏覽器類型（User-Agent）
  • 螢幕解析度
  • 語言設定

蒐集目的

上述資料僅用於網站流量分析與改善使用體驗，不用於識別特定個人。

IP 位址處理

您的 IP 位址會在伺服器端即時經過 SHA-256 雜湊處理，使用每日輪換的隨機鹽值（salt）。 雜湊後的值無法還原為原始 IP 位址。原始 IP 位址不會被儲存。

資料儲存與保留

  • 原始瀏覽紀錄：保留 90 天後自動刪除
  • 每日彙總統計（瀏覽量與不重複訪客數）：永久保留，不含個人資訊

資料處理於 Cloudflare Workers 基礎設施上，可能經由全球各地的節點處理。

本站託管於 GitHub Pages，GitHub 可能會蒐集伺服器日誌。 詳見 GitHub 隱私聲明。

第三方共享

本站蒐集的分析資料僅由網站管理者使用，不會提供、交換、出租或出售給任何第三方。

法律依據

依據 GDPR 第 6(1)(f) 條，本站基於「改善網站內容與使用體驗」之正當利益蒐集上述匿名資料。 依據台灣個人資料保護法第 8 條第 2 項第 6 款，本站之資料蒐集非基於營利目的，且對當事人顯無不利之影響。

您的權利

依據台灣個人資料保護法第 3 條及 GDPR 相關規定，您享有以下權利：

  • 查詢或請求閱覽您的個人資料
  • 請求補充或更正
  • 請求停止蒐集、處理或利用
  • 請求刪除

由於本站的分析資料已匿名化處理（IP 經不可逆雜湊），實務上可能無法識別特定個人的資料。 如有任何隱私相關疑問，歡迎透過 Email 與我聯繫。

不提供資料的影響

本站的分析為被動蒐集，在您瀏覽網頁時自動進行。 不提供上述資料不會影響您瀏覽本站的任何功能。

Manga TranslatorChrome 擴充功能

此擴充功能提供漫畫圖片的即時翻譯功能。除上述共用分析外，還涉及以下資料處理：

額外存取的資料

  • 當前頁面上的漫畫圖片內容（用於文字辨識）

第三方 API 傳輸

  • Google Cloud Vision API：圖片內容會傳送至 Google 進行 OCR 文字辨識
  • MyMemory Translation API：辨識出的文字會傳送至 MyMemory 進行翻譯

上述資料僅在您主動觸發翻譯功能時傳送，不會在背景自動蒐集。 傳送的資料不會由本擴充功能儲存或記錄。 第三方服務的資料處理方式請參閱各自的隱私權政策。

所需權限

  • 存取當前分頁的頁面內容（讀取圖片）
  • 連線至 Google Cloud Vision API 及 MyMemory API

Clean New TabChrome 擴充功能

此擴充功能提供自訂的新分頁頁面。除上述共用分析外：

資料處理

  • 使用 Chrome Storage API 儲存您的個人化設定（背景圖片、佈局偏好等）
  • 所有設定資料僅儲存於您的瀏覽器本機，不會傳送至任何外部伺服器

所需權限

  • storage：儲存使用者設定
  • unlimitedStorage：允許儲存較大的背景圖片檔案

Bot Command SyncWeb 應用

此工具提供 StreamElements 與 Nightbot 的指令同步管理。除上述共用分析外：

OAuth 與 API 資料

  • StreamElements：需要您提供 JWT Token 與 Channel ID，用於讀取及管理聊天機器人指令
  • Nightbot：透過 OAuth 2.0（Authorization Code + PKCE）授權，存取您的自訂指令

資料處理方式

  • 所有 Token 與授權資訊僅在您的瀏覽器中處理，不會傳送至本站伺服器
  • API 請求直接從您的瀏覽器發送至 StreamElements / Nightbot 的伺服器
  • 本站不儲存、記錄或轉發您的任何授權憑證

政策更新

本隱私權政策可能隨時更新。更新時將修改頁面頂端的「最後更新」日期。 建議您定期查閱本頁面以了解最新的隱私保護措施。
