# Design Philosophy: Mars Liu Personal Site

## 美學運動：「Quiet Machine」— 靜謐機械

### 核心理念

精密儀器的沉靜自信。不喧嘩、不炫技，但每一個細節都經過校準。
像一台高端音響的鋁合金面板——你不會注意到它的設計，但你會感受到它的質感。

---

### 1. 色彩語言：冷鍛金屬

不是純黑，不是純白。是介於兩者之間的**金屬灰階**——帶有微妙藍調的冷灰色，
像拉絲鋁、像鎢鋼、像清晨霧中的混凝土。

```
背景層（由深到淺）：
  基底：    #0f1114  — 近黑，帶冷藍底調（夜空）
  表面：    #1a1d23  — 主要內容區塊（碳纖維）
  浮層：    #242830  — 卡片、互動元素（石墨）

文字層（由亮到暗）：
  主文字：  #e4e7ec  — 高對比但不刺眼（磨砂銀）
  副文字：  #8b92a0  — 次要資訊（氧化鋁）
  微文字：  #525866  — 標籤、時間戳（暗合金）

強調色：
  焦點光：  #6b8afd  — 冷調靛藍，用於連結、光暈、互動反饋
  焦點光（低透明度）：rgba(107, 138, 253, 0.08) — 滑鼠追蹤光暈
  焦點光（中透明度）：rgba(107, 138, 253, 0.15) — hover 狀態

禁區：
  - 不使用純白 #ffffff（太刺眼）
  - 不使用純黑 #000000（太死板）
  - 不使用暖色調（不符合機械感）
  - 強調色只出現在互動元素上，靜態內容維持灰階
```

### 2. 空間與呼吸：精密間距

極簡不是「把東西拿掉」，而是「讓留下的東西都有足夠的空間存在」。

- **大區段之間**：至少 120px 間距，讓視線有停頓的餘裕
- **內容最大寬度**：800px，不需要填滿螢幕，窄一點讀起來更舒服
- **卡片內 padding**：慷慨的 32-40px，內容不擠壓邊框
- **元素之間**：用 24-32px 的一致節奏，像呼吸一樣規律

留白本身就是設計元素。一個段落結束後，下一個段落開始前的那段空白，
就像樂句之間的休止符——它讓內容有了節奏。

### 3. 字型系統：機械精度 × 人文溫度

兩種字型的對話，一個冷靜一個溫柔：

```
標題字型：   Inter（或 DM Sans）— 幾何無襯線，工程圖紙的精確感
等寬字型：   JetBrains Mono — 程式碼、標籤、技術細節
```

- **Hero 標題**：2.5-3rem，font-weight 600，字距微微加寬（letter-spacing: 0.02em）
- **區段標題**：1.25rem，全大寫，超寬字距（letter-spacing: 0.15em），像儀器面板上的刻字
- **正文**：1rem，行高 1.75，讓文字有空間呼吸
- **技術標籤**：JetBrains Mono，0.75rem，微微透明，像蝕刻在金屬上的型號標記

### 4. 光與影：滑鼠追蹤光暈

這是整個設計中唯一的「動態科技感」來源——滑鼠移動時，
一圈柔和的靛藍光暈跟隨游標，照亮附近的內容。

```
光暈規格：
  形狀：   radial-gradient，圓形
  半徑：   約 300-400px
  顏色：   rgba(107, 138, 253, 0.06) → transparent
  效果：   滑鼠經過的區域微微亮起，像手電筒掃過金屬表面
  性能：   用 CSS custom properties 追蹤座標，避免重繪整個頁面
```

光暈不是主角，它是氛圍。使用者不會刻意注意到它，
但如果拿掉，整個頁面就會感覺少了什麼——像是少了一層空氣。

**卡片互動**：滑鼠靠近卡片時，卡片邊框出現微妙的光邊（border 漸層），
像金屬邊緣在燈光下反射出的那一絲光線。

### 5. 動態節奏：滾動揭露

內容不是一次全部出現，而是隨著滾動逐步「浮現」：

```
進場動畫：
  方式：   opacity 0→1 + translateY(20px→0)
  時長：   600ms
  曲線：   cubic-bezier(0.16, 1, 0.3, 1) — 快進慢出，像精密機械的阻尼感
  觸發：   Intersection Observer，元素進入 viewport 20% 時啟動
  順序：   同一區段內的多個元素，依序延遲 80ms，產生「逐一就位」的效果
```

不要彈跳、不要旋轉、不要縮放。只有最克制的上移 + 淡入，
像霧氣散去後，物件安靜地出現在原位。

### 6. 構成原則：資訊的層級

```
Hero 區
  ── 名字（大，但不誇張）
  ── 一句話定位（副標題）
  ── 極簡的社群連結圖示

區段標題
  ── 全大寫 + 寬字距 + 細分隔線
  ── 像書的章節標題，安靜地宣告「這裡開始新的內容」

卡片
  ── 背景比頁面底色稍亮一階（#242830）
  ── 1px 邊框，顏色極淡（rgba(255,255,255,0.06)）
  ── hover 時邊框亮起（強調色低透明度）
  ── 沒有圓角或極小圓角（4px），機械感不需要柔軟

時間軸（Experience）
  ── 左側細線 + 節點，線的顏色是最暗的灰
  ── 節點是小圓點，hover 時亮起強調色
  ── 簡潔、克制，像電路板上的走線
```

### 7. 紋理與細節

- **背景噪點**：極微弱的 noise texture 覆蓋在背景上（opacity 2-3%），
  讓純色背景有「材質感」，像金屬表面的微觀紋理
- **分隔線**：不用粗線，用 1px + 極低透明度的白色（rgba(255,255,255,0.06)），
  若有似無，像金屬板材之間的接縫
- **游標**：保持系統預設，不做自定義游標（過度設計的標誌）

### 8. 禁止清單

這些東西不會出現在這個設計中：

- 漸層色背景（太花俏）
- 圓潤的大圓角卡片（不符合機械感）
- 彩色 icon（全部用單色）
- 裝飾性插圖或圖片（除非是專案截圖）
- 視差滾動（效果好但容易暈）
- 打字機效果（太 cliche）
- 粒子背景（效能殺手且 2020 年代已過時）
- 任何「跳出來」的動畫（彈跳、搖晃、旋轉）

---

## 設計哲學總結

> 「Quiet Machine」的核心信念：**真正的精緻是察覺不到的。**
>
> 使用者不會說「這個網站設計好棒」，
> 他們只會覺得「這個人的東西看起來很專業、很舒服」。
> 設計退到背後，讓內容和這個人本身成為焦點。
>
> 每一個像素都有存在的理由，每一毫秒的動畫都經過調校，
> 但這些努力的痕跡，應該像精密機械的內部齒輪——
> 完美運轉，卻藏在拉絲鋁的外殼之下。
