# Design Philosophy: Mars Liu Personal Site

## 美學運動：「Quiet Machine」— 靜謐機械

### 核心理念

精密儀器的沉靜自信。不喧嘩、不炫技，但每一個細節都經過校準。
像一台高端音響的拉絲鋁合金面板——你不會注意到它的設計，但你會感受到它的質感。

---

### 1. 色彩語言：拉絲銀灰

不是深黑，不是純白。是帶有冷藍底調的**銀灰金屬色**——
像陽極氧化鋁、像 MacBook 的 Space Gray、像精密加工後的不鏽鋼。

質感來自兩層效果的疊加：
1. **漸層底色** — 模擬光線在金屬曲面上的反射（中段偏亮、兩端偏暗）
2. **拉絲紋理** — 用 repeating-linear-gradient 模擬金屬加工後的細微刮痕紋路

```
背景結構（由下到上）：
  底層漸層：linear-gradient(180deg, #bcc0c8 0%, #d8dce4 40%, #e2e5eb 60%, #c8ccd4 100%)
            — 上暗 → 中亮 → 下暗，模擬弧形金屬板的光線反射
  拉絲紋理：三組不同間距的 repeating-linear-gradient 疊加
            — 間距 3px / 5px / 7px，半透明黑白交替
            — 視覺上產生隨機的水平刮痕，像真實的拉絲鋁

基底色（用於 neumorphism 計算基準）：
  中段色：  #d0d4dc  — 漸層的中間值
  陰影暗面：#adb1b9  — 金屬凹陷處的暗影
  陰影亮面：#edf1f9  — 金屬凸起處的高光

文字色階：
  主文字：  #2a3040  — 深灰藍（雷射雕刻）
  副文字：  #505868  — 中灰（蝕刻標記）
  微文字：  #808898  — 淺灰（模糊刻印）

強調色：
  焦點光：  #5a7aed  — 冷調靛藍
  光暈：    rgba(90, 122, 237, 0.10) — 滑鼠追蹤光暈
  hover：   rgba(90, 122, 237, 0.08) — 卡片內部跟蹤光

規則：
  - 不使用純白或純黑
  - 不使用暖色調
  - 強調色只出現在互動回饋上（hover、active），靜態內容維持灰階
  - 所有「顏色」都來自光影，不來自色彩本身
```

### 2. 觸覺語言：Neumorphism 的凸起與按壓

這是整個設計的核心特徵——元素從金屬表面「浮起」，
hover 時「按入」表面，產生實體按鈕的觸覺回饋感。

```
凸起陰影（raised）：
  大：  5px 5px 10px #adb1b9, -5px -5px 10px #edf1f9
  小：  3px 3px 6px #adb1b9,  -3px -3px 6px #edf1f9

按壓陰影（pressed）：
  inset 3px 3px 6px #adb1b9, inset -3px -3px 6px #edf1f9

適用元素：
  - 專案卡片（raised → hover pressed）
  - 技能標籤（raised-sm → hover pressed）
  - 社群圖示（raised-sm → hover pressed）
  - 時間軸節點（raised-sm → hover 強調色光暈）
  - 導覽列漢堡選單（raised-sm → hover pressed）

不適用：
  - 區段標題（文字即可，不需要容器）
  - 正文段落（文字流不需要觸覺）
  - 時間軸主線（是結構線，不是按鈕）
```

### 3. 空間與呼吸：精密間距

極簡不是「把東西拿掉」，而是「讓留下的東西都有足夠的空間存在」。

- **大區段之間**：至少 120px 間距（padding: 5rem 0），讓視線有停頓的餘裕
- **內容最大寬度**：800px，不需要填滿螢幕，窄一點讀起來更舒服
- **卡片內 padding**：24px，內容不擠壓邊框但也不空曠
- **元素之間**：用 24-32px 的一致節奏，像呼吸一樣規律
- **Hero 區**：上下各 6rem / 5rem，給名字足夠的尊嚴空間

### 4. 字型系統：工程圖紙的精確感

```
標題 / 正文：Inter — 幾何無襯線，工程師的字型
技術標記：    JetBrains Mono — 等寬，程式碼的母語
```

- **Hero 標題**：2.75rem，weight 600，letter-spacing: 0.02em
- **Hero 副標題**：JetBrains Mono，0.9rem，全小寫，`//` 分隔符，像程式碼註釋
- **區段標題**：JetBrains Mono，0.7rem，全大寫，letter-spacing: 0.15em，像儀器面板上的蝕刻標記
- **正文**：Inter，1rem，行高 1.75
- **時間戳 / 技術標籤**：JetBrains Mono，0.75rem，muted 色

### 5. 光與影：滑鼠追蹤光暈

滑鼠移動時，一圈柔和的靛藍光暈跟隨游標。
在銀灰背景上，光暈效果比深色底更微妙——像光線掠過金屬表面的反射。

```
光暈規格：
  半徑：   250px（比深色底小，因為淺色背景上太大會失焦）
  顏色：   rgba(90, 122, 237, 0.10) → transparent
  z-index：1（在噪點紋理之上，內容之下）
```

**卡片內部光暈**：滑鼠在卡片內移動時，卡片內出現跟蹤滑鼠的淡色光斑。
與 neumorphism 的按壓效果疊加，hover 時既按下去又有光跟著走。

### 6. 動態節奏：滾動揭露

```
進場動畫：
  方式：   opacity 0→1 + translateY(20px→0)
  時長：   600ms
  曲線：   cubic-bezier(0.16, 1, 0.3, 1) — 快進慢出，精密機械的阻尼感
  觸發：   Intersection Observer，元素進入 viewport 15% 時啟動

逐一就位（stagger）：
  同一區段的子元素依序延遲 60ms 進場
  適用於：技能標籤、專案卡片

無障礙：
  prefers-reduced-motion: reduce 時，所有動畫取消，元素直接顯示
```

### 7. 紋理與細節

- **拉絲紋理**（取代噪點）：三組 repeating-linear-gradient 疊加，模擬 CNC 加工後的金屬表面
- **區段分隔**：1px 實線，顏色用陰影暗面色（#adb1b9），若有似無
- **圓角**：10-14px，比直角柔軟但不會變成「可愛」，保持精密感
- **游標**：系統預設，不做自定義

### 8. 構成原則

```
Hero 區
  ── 名字（2.75rem，居中）
  ── 副標題（mono，小寫，// 分隔）
  ── 簡介段落（窄寬度，置中）
  ── 社群圖示（neumorphism 小按鈕，一排居中）

區段標題
  ── JetBrains Mono 全大寫 + 超寬字距
  ── 底部 1px 分隔線
  ── 像儀器面板上的標記，安靜地劃分區域

專案卡片
  ── 2 欄網格（手機 1 欄）
  ── neumorphism 凸起 → hover 按壓 + 內部追蹤光
  ── 類別標籤用強調色 mono 字型
  ── 只放有公開連結的專案

時間軸
  ── 左側 2px 細線 + 凸起小圓點
  ── hover 時圓點亮起強調色
  ── 工作資訊的層級：年份 → 公司 → 職位 → 描述
```

### 9. 禁止清單

- 漸層色背景
- 彩色 icon
- 裝飾性插圖
- 視差滾動
- 打字機效果
- 粒子背景
- 彈跳 / 搖晃 / 旋轉動畫
- 連結到不存在或 private 的頁面

---

## 設計哲學總結

> 「Quiet Machine」的核心信念：**真正的精緻是察覺不到的。**
>
> 銀灰的金屬表面上，元素安靜地浮起；你的手指靠近，它們溫順地凹下。
> 一圈淡藍的光暈跟隨你的視線移動，但你不會注意到它——
> 你只會覺得「這個人做的東西，看起來很乾淨、很專業」。
>
> Neumorphism 不是為了炫技，是為了觸覺。
> 光暈不是為了酷炫，是為了氛圍。
> 每一個設計決策都服務於同一個目標：讓內容成為焦點，讓設計退居幕後。
